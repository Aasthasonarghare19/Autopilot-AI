Autopilot AI- Academic Workflow Assistant

📌Description
Autopilot AI is a web based application that automates academic tasks. It helps students generate a plan, detailed report, ppt and summary from a given topic.

🚀Features
- Generate academic plan
- Creates detailed report
- Automatically generates ppt
- Provides summary
- Easy to Use Interface

🛠️Technology Used
- Python
- Streamlit
- python-pptx

⚙️How to Run Locally
- Install required libraries
- Run: Streamlit run app.py
- Open the link in browser
